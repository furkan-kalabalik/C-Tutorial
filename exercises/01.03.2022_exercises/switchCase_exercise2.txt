Write a C program to find all roots of a Quadratic equation now using switch case

Quadratic equation = ax^2+bx+c

INPUTS
-------
Take constants of quadratic equation from user --- > a, b, c
float a, b, c

OUTPUTS
-------
float root1, root2, imaginary

TEST CASE
----------
INPUTS							OUTPUTS
a = 1, b = 2, c = 1				root1 = root2 = -1

INPUTS							OUTPUTS
a = 1, b = 1, c = -6			root1 = -3 root2 = 2
